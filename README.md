**如果你只花 20 秒看这个仓库，就先看这四句：**

- `145%不是数字，是先把全世界拖到他的报价单上`
- `杜蕾斯不靠露骨成交，靠先把尴尬拿走`
- `家里没矿别碰金融，不是毒舌，是先替一个家庭算损益表`
- `李佳琦卖的不是便宜，是“不买你今天就吃亏了”`

`persona-autopsy-skill` 不是在整理名言，也不是在给人物做性格测试。

它做的是另一件事：

**把一句话怎么变成场面、价格、身份、欲望、成交、退场权，拆回机器。**

你给我一个公众人物、品牌账号、创作者、销售主播，或者一段你主动提供的聊天人格。  
我会把里面最值钱的那套东西拆出来：

- 他怎么进场
- 他怎么抢框架
- 他怎么让不同人听见不同意思
- 他拿什么换注意力、顺从、信任、购买
- 他什么时候像神，什么时候会露馅
- 如果你要借这套东西，应该借哪一层，别碰哪一层

你给我材料，我给你三样东西：

- `Autopsy Report（解剖报告）`
  - 不是评价他“厉害不厉害”，而是拆清楚他靠什么持续赢注意力和判断权
- `Imitation Guide（模仿指南）`
  - 不是教你背口头禅，而是告诉你该偷哪套结构，哪里一学就假
- `Immersive Dialogue Mode（沉浸对话模式）`
  - 分析完成后，可选一次临时扮演，验证这套人格 machinery 到底会怎么开口

## 这仓库到底在拆什么

它不只拆“表达风格”，它拆的是**表达背后的权力结构**。

| 层 | 它真正想问的问题 | 最终会被拆成什么 |
| --- | --- | --- |
| `Hook Surface` | 为什么他一开口你就知道“这是谁” | 起手句型、数字/标签/双关、记忆钉子 |
| `Entry Stance` | 他是怎么进场的 | 裁判、老板、内行人、过来人、坏笑旁观者 |
| `Scene Control` | 他怎么让全场按他的框架讨论 | 抢定义权、改问题、拖别人到自己的桌子上 |
| `Narrative Engines` | 他总把事情讲成什么故事 | 赢家/输家、救赎、复仇、体面、现实、欲望 |
| `Judgment / Trade Logic` | 他到底在拿什么衡量世界 | 钱、地位、效率、忠诚、阶层、情绪、面子 |
| `Audience Split` | 他对不同人其实在说什么 | 对粉丝、路人、媒体、对手、平台的分层话术 |
| `Emotion Thermostat` | 他怎么调现场温度 | 煽动、安抚、暧昧、羞耻、 urgency、优越感 |
| `Pressure / Conversion Moves` | 他怎么让人转发、服气、下单、跟随 | 锚点、损失厌恶、身份诱导、共识伪造、关系缓冲 |
| `Defense / Exit Moves` | 他失手时怎么退 | 改定义、换战场、甩锅、装轻描淡写、拿小胜当大胜 |
| `Weapon Library` | 他最常用的武器是什么 | 高爆词、半句留白、标签命名、狠话开场、伪共识句 |
| `Cracks / Failure Modes` | 这套东西什么时候会突然失灵 | 过度表演、平台错位、证据不足、 audience 反噬 |

一句话总结：

**我们拆的不是“他像什么人”，而是“他怎么把一句话变成别人必须回应的现实”。**

## 你会拿到什么

### 1. `Autopsy Report（解剖报告）`

你拿到的不会是“这个人很会说话”“这个品牌很会营销”这种空赞美。

你会拿到一份可以直接拿去复盘、借用、拆解的结构报告，默认至少覆盖：

- `一句话判词`
- `Hook Surface`
- `Entry Stance`
- `Scene Control`
- `Expression DNA`
- `Narrative Engines`
- `Judgment / Trade Logic`
- `Audience Split`
- `Emotion Thermostat`
- `Pressure / Conversion Moves`
- `Defense / Exit Moves`
- `Weapon Library`
- `Cracks, Contradictions, Failure Modes`

### 2. `Imitation Guide（模仿指南）`

不是“模仿他的声音”，而是告诉你：

- 最值得偷的是哪几招
- 哪一层碰了就廉价
- 句子该怎么起手、怎么收口
- 情绪热度该到哪里
- 什么地方该留白
- 哪些动作看似像，其实是低仿翻车

### 3. `Immersive Dialogue Mode（沉浸对话模式）`

如果你明确要求，可以在分析完成后进入一次临时扮演：

- 只做临时扮演
- 不导出 skill
- 不保存为人格资产
- 不把“像”伪装成“复活”

## 分析维度

现在这个 skill 默认不是 5 个切面，而是一套更完整的剖面。

### `1. 一句话判词`

先给一个能抓住全局的 verdict。不是玩金句，而是快速给出这套人格最值钱的发动机。

### `2. Hook Surface`

先看它最容易被记住的那一层：

- 极端数字
- 双关和半句留白
- 狠话开场
- 标签命名
- 伪共识句式

### `3. Entry Stance`

他是怎么进门的？

- 裁判
- 老板
- 过来人
- 兄弟 / 闺蜜 / 军师
- 懂分寸的品牌旁白

### `4. Scene Control`

真正厉害的人，不是会说，而是会改现场。

这里会拆：

- 他怎么重新定义问题
- 他怎么抢结论权
- 他怎么逼别人站到自己的地板上说话

### `5. Narrative Engines`

每个强人格都在重复几个故事模板。

比如：

- 世界在坑我们，我来讨债
- 你现在不算账，未来会出血
- 我不是来卖货，我是来替你避免尴尬
- 你不是买了便宜，你是逃过了损失

### `6. Judgment / Trade Logic`

这层是最值钱的。

它回答：

- 他真正拿什么当尺子
- 他最在乎的是注意力、效率、面子、现金流还是控制权
- 他嘴上讲价值，底层到底在做什么交易

### `7. Audience Split`

很多人格最厉害的地方，不是说一句所有人都爱听，而是**一句话让不同人各自听见不同东西**。

这层会拆：

- 粉丝听到的承诺
- 路人听到的热闹
- 对手听到的威胁
- 平台和媒体听到的信号

### `8. Emotion Thermostat`

他怎么调温度：

- 让人热血
- 让人羞耻
- 让人安心
- 让人欲望上升
- 让人觉得“现在就该做点什么”

### `9. Pressure / Conversion Moves`

这是“成交力”真正待着的地方。

这里会拆：

- 怎么开高锚点
- 怎么制造损失厌恶
- 怎么先给身份，再给产品
- 怎么把转发和购买变成“顺手”

### `10. Defense / Exit Moves`

真正高级的不只是能上桌，还要能退场。

会看：

- 他怎么甩锅
- 他怎么把回撤改写成赢
- 他怎么装作自己根本没下重注

### `11. Weapon Library`

把他的常用武器单独列出来：

- 高频词
- 常见句型
- 开场动作
- 留白手法
- 伤害性最强的判断模板

### `12. Cracks, Contradictions, Failure Modes`

最后必须把神话拆坏一点。

否则你学到的只会是 fan fiction，不是可用结构。

## 它怎么避免只靠一条梗瞎编

这不是“看两个切片就开始装懂”的 skill。

默认会看材料等级：

- `Grade A`
  - 长演讲、长访谈、本人公开账号、长文、系统性表达
- `Grade B`
  - 短采访、短视频、广告物料、公开切片
- `Grade C`
  - 二手分析、媒体综述、整理稿
- `Grade D`
  - 你主动提供的截图、聊天片段、转录
- `Grade E`
  - 梗图、失真转述、只剩 meme 的公共印象

材料不足，就降级成 `弱拆解`。  
不装作自己已经把一个人看透。

## 长案例

下面这些不是“模仿得像不像”的展示。  
它们在展示：**这把刀能切出多少层。**

### 特朗普：145%不是数字，是先把全世界拖到他的报价单上

一句话判词：

**他最会卖的不是政策，是让所有人先围着他的价码说话。**

你会看到：

- 极端数字不是测算，是锚点
- 政策不是内容，是交易舞台
- 媒体盯真假，他盯谁被迫回应
- 回撤不叫回撤，叫重新定义已经赢了哪一半

完整长案例见 [`examples/public-figure-autopsy.md`](./examples/public-figure-autopsy.md)。

### 杜蕾斯：卖的不是套，是把尴尬先拿走

一句话判词：

**它先把空气调对，再让产品像顺手出现。**

你会看到：

- 它不是在讲性，是在处理“怎么开口”
- 它不是先卖安全，而是先救气氛
- 它真正的成交动作，是让用户愿意转发那句文案
- 它最强的地方不是敢说，而是刚好停住

完整长案例见 [`examples/brand-account-autopsy.md`](./examples/brand-account-autopsy.md)。

### 张雪峰：家里没矿别碰金融，不是毒舌，是先替一个家庭算损益表

一句话判词：

**他拿狠话当手电筒，不是为了刺痛你，是为了让一个家庭先看见未来哪儿要出血。**

你会看到：

- 他先打碎浪漫叙事，再把问题拖回现金流
- 他真正卖的是“别走错一步”的安全感
- 他不是只在说专业，他在说阶层门槛
- 他的一切爆句，底下都有一张家庭 ROI 表

完整长案例见 [`examples/creator-autopsy.md`](./examples/creator-autopsy.md)。

## 快速短切样本

除了三份长案例，仓库里还放了一组短切样本，专门展示不同赛道的人设 machinery：

- 李佳琦：不是便宜，是把“不买就亏了”演成现场直播
- 罗永浩：不是毒，是先自黑把场子换成自己的，再反手开刀
- 马斯克：不是狂，是把未来当 deadline 发给今天
- 罗翔：不是普法，是把道德困境搭成戏剧舞台再下判词
- 苹果发布会：不是讲参数，是把参数翻译成一种更高级的生活秩序
- 咪蒙：不是只会煽动情绪，而是把委屈翻译成可转发的立场

完整短切样本见 [`examples/rapid-cuts.md`](./examples/rapid-cuts.md)。

## 边界

这把刀可以很锋利，但它不是无边界的。

- 不替真实人物背书
- 不鼓励上传不归你控制的私密材料
- 材料不足时只做弱拆解，不硬装深刻
- 临时扮演只是验证分析，不是永久封装人格
- 不直接导出新的人格 skill package
