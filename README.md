**如果你只花 20 秒看这个仓库，就先看这三句：**

- `145%不是数字，是开价`
- `杜蕾斯卖的不是套，是开口那一秒不尴尬`
- `家里没矿别碰金融，不是毒舌，是筛选器`

`persona-autopsy-skill` 专拆三件事：

- `钩子`
  - 为什么这一句会被记住
- `价码`
  - 这句话真正想换来的是什么
- `杀伤力`
  - 它怎么让人上头、破防、买单、转发

你给我一个公众人物、品牌账号、创作者，或者一段聊天人格。  
我把里面**最会带节奏、最会抬价、最会成交**的那部分拆成结构。

你给我材料，我给你三样东西：

- `Autopsy Report（解剖报告）`
  - 这套人格是怎么运作的
- `Imitation Guide（模仿指南）`
  - 哪些能学，哪些别学，一学就假
- `Immersive Dialogue Mode（沉浸对话模式）`
  - 分析完成后，可选一次临时扮演

## 它能拆什么

| 对象 | 常见材料 | 最终会拆出的东西 |
| --- | --- | --- |
| 公众人物 | 采访、演讲、辩论、播客、社媒发言 | 气势结构、叙事主轴、攻防动作、露馅点 |
| 品牌 / 官号 / 创作者账号 | 帖子、评论、短视频文案、回复风格 | 平台腔调、角色设定、包袱、转化动作 |
| 你主动提供的聊天人格 | 聊天记录、截图、群聊片段、语音转录 | 控场习惯、缓冲词、压人方式、关系姿态 |

材料足够，就做完整解剖。  
材料不够，就做“弱拆解”，不装作自己已经看穿了一切。

## 你会拿到什么

### 1. `Autopsy Report（解剖报告）`

它不是“这个人很自信 / 很会说话”这种空形容词。

它会把人格拆成可复用的结构：

- `Expression DNA`
  - 这人一开口，句子是怎么站起来的
- `Narrative Routines`
  - 他喜欢把事情讲成什么故事
- `Judgment Frames`
  - 他默认按什么标准做判断
- `Manipulation / Defense Moves`
  - 他怎么反击、怎么防守、怎么推卸、怎么控场
- `Cracks, Contradictions, Failure Modes`
  - 他在哪些场景里会突然失效、变假、翻车

### 2. `Imitation Guide（模仿指南）`

不是“模仿他的口头禅”，而是告诉你：

- 该学哪一层
- 不该碰哪一层
- 句子应该怎么起手
- 情绪应该热到什么程度
- 什么动作一做就从“像”变成“廉价模仿”

### 3. `Immersive Dialogue Mode（沉浸对话模式）`

如果你明确要求，可以在分析完成后进入一次临时扮演：

- 只做临时扮演
- 不导出 skill
- 不保存为人格资产
- 不声称“已经复活了一个人”

## 效果示例

下面这些例子不是在卖“模仿得像”。  
它们在展示这把刀到底能切到哪里。

### 例子一：145%不是数字，是开价

目标：Donald Trump（特朗普）

一句就够记住他：

- **退不叫退，叫重新定义赢。**
- **先把天花板抬飞，再让所有人围着你的数字谈。**

你可能会问：

- “为什么他说离谱数字时，听上去不像算错，更像在开价？”
- “为什么他经常反复横跳，却还是能把自己包装成赢？”

解剖结果会长这样：

```markdown
## 解剖报告（Autopsy Report）

### 判断框架
- 万物皆交易，政策也是交易
- 数字首先是谈判锚点，其次才是经济测算
- 真正重要的不是客观输赢，而是谁能公开宣称赢

### 操控 / 防御动作
- 先把开价拉到离谱，逼所有人围着他的锚点谈
- 用不可预测性拖高对手的防守成本
- 如果事实不占优，就改写胜利标准
- 出现市场压力或象征性让步时，会给自己找一个能宣布赢的退场口

## 模仿指南（Imitation Guide）

### 该学什么
- 先定义桌子，再谈条件
- 把夸张当 framing 工具，而不是单纯吹牛
- 给自己预留一个可以公开宣布胜利的退场口

### 不该学什么
- 只学 insult、大写字母和音量
- 忘了他其实一直在做交易，不是在单纯发泄
- 没有筹码时硬装 unpredictable
```

临时扮演预演：

```text
用户：145%的关税不是太高了吗？
临时扮演：太高？一点也不。那不是终点，那是开价。你先把天花板抬上去，所有人都会回来找你谈。现在桌子是你的了。

用户：媒体说你又改口了。
临时扮演：改口？不是改口，是交易。只要最后他们坐到我的数字上谈，我就已经赢了第一轮。
```

完整示例见 [`examples/public-figure-autopsy.md`](./examples/public-figure-autopsy.md)。

### 例子二：杜蕾斯卖的不是套，是开口那一秒不尴尬

目标：杜蕾斯

一句就够记住它：

- **真正高级的情趣文案，不是把性写脏，是把尴尬写没。**
- **它卖的不是产品，是“现在这句可以怎么说出口”。**

你可能会问：

- “为什么它写的是安全套，大家读出来的却是气氛、暧昧和时机？”
- “为什么别的品牌写两性像科普，它写两性像会被收藏的梗？”

解剖结果会长这样：

```markdown
## 解剖报告（Autopsy Report）

### 叙事套路
- 产品不先出现，先出现的是场景和关系张力
- 文案真正卖的不是“安全”，而是“不扫兴”
- 梗感停在暧昧前半步，让读者自己补完最后那半句

### 破绽与失效点
- 一旦把双关写满，就会从高级变油腻
- 一旦借势慢半拍，就会从机灵变成跟风

## 模仿指南（Imitation Guide）

### 该学什么
- 先写关系，再写产品
- 让读者补完最后半句
- 把分寸感当成真正的文案能力

### 不该学什么
- 低级黄梗
- 医疗说明书式说教
- 为了借势而借势
```

临时扮演预演：

```text
用户：为什么你们文案总是不把话说满？
临时扮演：因为真正让人心动的，不是被说破，是差那半句。

用户：那产品到底什么时候出现？
临时扮演：气氛刚好，尴尬消失，产品自然就该出现了。太早像打断，太晚像补救。
```

完整示例见 [`examples/brand-account-autopsy.md`](./examples/brand-account-autopsy.md)。

### 例子三：家里没矿别碰金融，不是毒舌，是筛选器

目标：张雪峰

一句就够记住他：

- **他不是在劝退专业，他是在给家庭做止损。**
- **先谋生，再谋热爱，是他所有狠话的底板。**

你可能会问：

- “为什么他每次开口都像在泼冷水，但家长和学生还是会疯狂转发？”
- “为什么他的狠话听上去像段子，底下其实是一整套 ROI 和阶层流动逻辑？”

解剖结果会长这样：

```markdown
## 解剖报告（Autopsy Report）

### 判断框架
- 专业选择不是兴趣测验，是家庭资源配置
- 先看兜底能力，再看理想表达
- 一切听起来很浪漫的专业，最后都要落到就业和现金流

### 操控 / 防御动作
- 先用一句狠话把幻想打穿
- 再用非常具体的就业后果把人拖回现实
- 用“家里有没有矿”“能不能托底”这种说法快速建立筛选器

## 模仿指南（Imitation Guide）

### 该学什么
- 把抽象选择翻译成现实后果
- 用具体岗位、城市、收入、家庭条件落地
- 让残酷感服务于判断，不是服务于表演

### 不该学什么
- 为狠而狠
- 只会吓人，不会给路径
- 把个例包装成永恒铁律
```

临时扮演预演：

```text
用户：我就想学新闻，行不行？
临时扮演：行，当然行。问题不是行不行，问题是毕业以后谁给你兜底。热爱我不反对，账你得先算明白。

用户：那金融呢？
临时扮演：家里没矿别碰金融。不是这个专业不好，是你进去以后拼的很多时候不是卷子，是门槛外面的东西。
```

完整示例见 [`examples/creator-autopsy.md`](./examples/creator-autopsy.md)。

## 它明确不做什么

- 不生成新的 `SKILL.md`
- 不永久封装一个可复用的人格资产
- 不在证据不足时脑补“完整人格”
- 不替用户保存大体量私密原始语料

一句话：**它不是为了夸一个人会说话，而是为了拆清楚那套让别人买账的机制。**

## 默认触发词

- “拆一下这个人设”
- “分析这个账号怎么说话”
- “这个人的套路是什么”
- “给我这个人格的模仿指南”
- “让我和这个人设对话”

## 默认流程

1. 识别对象类型
2. 判断来源类型与可信度
3. 输出 `Autopsy Report`
4. 输出 `Imitation Guide`
5. 如果你明确要求，进入 `Immersive Dialogue Mode`

## 边界

- 只分析公开材料，或用户主动提供的半公开材料
- 不鼓励用户上传不受自己控制的私密数据
- 不把公开人设等同于完整人格
- 材料太少、风格冲突或来源太弱时，只输出弱拆解
- 高拟真扮演只在证据足够且用户明确要求时启用
